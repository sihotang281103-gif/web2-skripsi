Tentang Website

Website ini dikembangkan sebagai objek penelitian skripsi dengan judul:

Studi Perbandingan Kecepatan dan Keamanan Website Web 2.0 versus Web3 Menggunakan Cloudflare CDN dan TLS 1.3.

Teknologi yang Digunakan

  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap 5
  • Cloudflare CDN
  • TLS 1.3
  • Vercel Hosting
  • IPFS (Pinata)

Tujuan Penelitian

Penelitian ini bertujuan membandingkan tingkat kecepatan dan keamanan antara website berbasis Web 2.0 dan Web3 dengan menggunakan parameter TTFB, Largest Contentful Paint (LCP), Total Load Time, SSL/TLS Grade, Security Headers, dan Vulnerability Assessment menggunakan OWASP ZAP.
